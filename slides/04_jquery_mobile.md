# Pros #
* jQuery
* Atributos `data-` de HTML5 (Mayor semántica)
* UI imita diseño iOS.

!SLIDE bullets
# Contras #
* Todavía en alfa.
* UI imita diseño iOS.
* UI relativamente limitada.
* No permite definir estilos propios fácilmente.
* No tiene acceso al hardware.
